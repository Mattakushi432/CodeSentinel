Privacy by Design

        
Your source code never leaves your server. All LLM inference runs locally via Ollama — zero external API calls required.

      


      

        

          
        

        
Local AI via Ollama

        
Powered by qwen2.5-coder:7b and any model available via Ollama. Runs on CPU with 8 GB RAM. Optional Groq fallback for burst load.

      


      

        

          
        

        
GitHub · GitLab · Gitea

        
Connect any repository from GitHub, GitLab, or Gitea. Reviews trigger automatically on every pull request via webhook.

      


      

        

          
        

        
Inline PR Comments

        
Findings are posted as inline comments on the exact lines of your PR — right inside your normal code review flow, no separate dashboard.

      


      

        

          
        

        
Custom Review Rules

        
Define per-repository rules that guide the AI: security checklists, coding standards, team conventions — all stored locally on your server.

      


      

        

          
        

        
Docker Compose Ready

        
One docker compose up -d starts the FastAPI app, Ollama inference engine, and Caddy reverse proxy. Works on any Linux server.

      


    

  










  

    

      
Workflow

      
Up and reviewing in minutes

      
Connect your repository once. Every pull request is reviewed automatically from that point on.

    


    

      

        
01

        
Connect Your Repository

        
Add your GitHub, GitLab, or Gitea repository in the dashboard. CodeSentinel generates a webhook URL — paste it into your repo and you're done.

      

      

        
02

        
AI Reviews Each PR

        
When a pull request opens, the webhook fires. Ollama runs the diff through your local model and identifies bugs, security issues, and style violations.

      

      

        
03

        
Inline Comments Appear

        
Findings are posted as inline comments on the exact lines of your PR — right inside your normal code review view. No separate dashboard to check.

      

    

  










  

    

      
Quick Start

      
Five commands to production

      
A single Docker Compose stack — FastAPI, Ollama, Caddy. No cloud account needed.

    


    

      

        bash
        
      

      

        
# 1. Clone the repository
git clone https://github.com/Mattakushi432/CodeSentinel.git
cd CodeSentinel

# 2. Configure environment
cp .env.example .env
# Set SECRET_KEY and BASE_URL in .env

# 3. Start all services
docker compose up -d

# 4. Pull the AI model  (~4 GB, one-time)
docker exec codesentinel_ollama ollama pull qwen2.5-coder:7b-instruct

# 5. Open the dashboard
open http://localhost:8000
